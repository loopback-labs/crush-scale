# Lovemeter

A playful love compatibility calculator (notebook-style UI) by [Merchandice](https://merchandice.in/). Live at [lovemeter.merchandice.in](https://lovemeter.merchandice.in/).

## Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

Requires Node.js and npm.

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Deploy

The repo includes a GitHub Actions workflow that builds and deploys to GitHub Pages. In the repository settings, set **Pages** → **Source** to **GitHub Actions**.

For a custom subdomain (e.g. `lovemeter.merchandice.in`), configure DNS and GitHub Pages custom domain as documented in [GitHub Pages custom domains](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site).
